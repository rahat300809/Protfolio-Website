# 🌟 Rahat's Personal Portfolio

A fully custom, visually stunning personal portfolio website built from scratch — designed to leave an impression. From liquid glass buttons to animated shader backgrounds, every detail was crafted with intention.

---

## ✨ What's Inside

### 🏠 Hero & About
- Animated beams background with a cinematic entrance
- Personal intro with smooth scroll interactions
- Skills showcased with animated icons and visual progress

### 💼 Work & Projects
- Project cards with modal previews, image galleries, and direct GitHub/live links
- Research section for academic & professional publications
- CV generator for instant downloadable resume

### 📅 Events & Life
- Carousel-based life events timeline
- Certificates gallery with clean card layout

### 📝 Blog
- Secure blog section with rich content support
- Video playback and media-rich posts

### 🛠️ Admin Dashboard
- Full admin panel to manage portfolio content dynamically
- Real-time updates via Firebase Firestore
- Upload projects, blog posts, stats, events, and social links on the fly

### 📬 Contact
- Functional contact form
- Social media links with animated icons

---

## 🎨 Design Highlights

- **Glassmorphism & liquid glass UI** — buttons feel like they're alive
- **Animated beam backgrounds** — SVG-powered cinematic visuals  
- **Shader-based backgrounds** — GPU-rendered ambient motion
- **Smooth framer-motion transitions** — every page feels buttery
- **Dark, premium aesthetic** — deep blacks, glowing accents, and subtle gradients
- **Fully responsive** — looks great on mobile, tablet, and desktop

---

## 🧰 Tech Stack

| Layer | Tech |
|-------|------|
| Framework | React 19 + TypeScript |
| Build Tool | Vite 6 |
| Styling | TailwindCSS v4 |
| Animations | Framer Motion, Three.js |
| Backend | Express + Firebase Firestore |
| Auth | Firebase Auth |
| Deployment | Firebase Hosting / Vercel |

## 📁 Project Structure

```
src/
├── components/       # Reusable UI components (beams, liquid glass, carousels...)
├── pages/            # Route-level pages (Home, Projects, Blog, Admin...)
├── context/          # Global state & Firebase context
├── data/             # Static data & content seeds
└── lib/              # Utilities & Firebase config
```
